# Task-04: Image-to-Image Translation using Pix2Pix GAN

## Objective
To implement an image-to-image translation model using Pix2Pix (Conditional GAN).

## Tools Used
- Python
- PyTorch
- Google Colab

## Dataset
CIFAR-10 dataset

## Concepts Used
- Generator
- Discriminator
- Conditional GAN

## Output
The model generates transformed images from input images.

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run training:
   python train.py
